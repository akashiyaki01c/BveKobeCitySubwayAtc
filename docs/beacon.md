# 地上子 仕様
2番目の引数は使用しません。

記述例
```
1000;	Beacon.Put(<Type>, 0, <Option>);
1100;	Beacon.Put(11, 0, 1); // 始発駅を谷上に設定
```

|Type|Option|概要|
|:--:|:--:|:--|
|0|1..=6|ATO地上子の設定(未実装)|
|11|0..=17(駅ID)|始発駅の設定|
|12|0..=17(駅ID)|行先の設定|
|13|0..=5(種別ID)|種別の設定|
|14|0..=99|運行番号の設定|
|15|0..=2|TIMS距離程プレフィックスの設定|
|16|..|TIMS距離程原点の設定|
|17|0..=1|TIMS距離程加減算の設定|
